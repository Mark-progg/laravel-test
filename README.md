# Laravel Test project

## Описание
Тестовое задание по созданию мини-бэкенда для админ панели Filament

## Стек
-Laravel 13
-PHP 8.3
-Filament 5.4.1
-PostgrsSQL 18

### Инструкция по запуску

```bash
git clone https://github.com/Mark-progg/laravel-test.git
cd laravel-test
make up
make install
```

- Для входа в админку нужно перейти в бразере по 
http://localhost:8000/admin
И воспользоваться этими учетными данными:
